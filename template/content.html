Illustrated jq tutorial

jq is a very versatile tool for working with structured information in json format, however it needs some time to get used to building processing pipelines with this tool. This tutorial covers several examples, each example comes with links that show you the intermediate steps of each processing pipeline. The html for this tutorial is generated by [this scrip](https://github.com/MoserMichael/jq-illustrated/blob/master/scr.sh)
